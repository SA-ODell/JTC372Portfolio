About This Site

This site was written using Atom, and uses icons and fonts from Font Awesome and Devicon. Google Fonts provided the Oswald and Bitter typefaces. Images were edited in Photos and sized in Preview.

The color scheme was checked for WGAC 2.0 compliance using WAVE, a web accessibility evaluation tool, and Contrast Ratio Checker, a Chrome extension.

It is hosted by GitHub Pages.
